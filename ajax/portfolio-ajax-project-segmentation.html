Customer Segmentation for Target Marketing

Project Info:

Provides techniques for clustering customers based on their transaction data so that a company can target different customer segments with different types of promotional offers.
The dataset is from the loyalty program of a former airline which holds the information on 3,999 customers.

Project Details:

  • Field:User Segmentation / Target Marketing
  • Technologies:R, K-Means / Hierarchical clustering
