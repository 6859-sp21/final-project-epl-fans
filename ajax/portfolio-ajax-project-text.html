Evaluating Sentiment Capturing of Text Summarization Models

Project Info:

As a course requirement of 'Applied Machine Learning', this project uses Natural Language Processing (NLP) techniques to accurately and concisely capture what people are saying in the user reviews. In particular, a machine learning pipeline was created to read and create summaries of key issues and sentiment embedded in the customer reviews.
